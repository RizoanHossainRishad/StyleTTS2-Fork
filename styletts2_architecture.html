③ Training Batch Loop (enumerate train_dataloader)
🔄 Forward Pass Pipeline
1text_aligner(mels, mask, texts) → soft s2s_attn → monotonic s2s_attn_mono
2text_encoder → ASR features (50% soft / 50% mono — stochastic augmentation)
3per-utterance loop → style_encoder (acoustic s) + predictor_encoder (prosodic s_dur)
4BERT + bert_encoder → duration feature d_en
5predictor(d_en, s_dur) → duration logits d, prosody p
6Random clip slice bounded by mel_len ≤ max_len/2 and shortest utterance
7pitch_extractor (frozen) → F0_real, N_real from GT clip
8predictor.F0Ntrain(p_en, s_dur) → F0_fake, N_fake
9decoder(en, F0_fake, N_fake, s) → y_rec | isnan(y_rec) → skip
DDiffusion sampler + EDM loss + style L1 (active from diff_epoch)
⚔ Discriminator Update
dl(wav.detach(), y_rec.detach()) → d_loss → backward
optimizer.step(msd) · optimizer.step(mpd)
🔧 Generator Loss (g_loss)
λ_mel·stft λ_F0·smoothL1 λ_ce·BCE λ_norm·smoothL1 λ_dur·L1 λ_gen·GAN λ_slm·WavLM λ_sty·L1 λ_diff·EDM λ_mono·L1×10 λ_s2s·CTC
Steps: bert_encoder, bert, predictor, predictor_encoder, style_encoder, decoder, text_encoder, text_aligner + diffusion (from diff_epoch)
🧬 SLM Adversarial from joint_epoch
slmadv() → d_loss_slm + loss_gen_lm
Adaptive grad clip via predictor norm threshold
Extra scale on lstm, duration_proj, diffusion params
